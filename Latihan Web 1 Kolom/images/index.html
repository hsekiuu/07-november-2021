What is Jquery?

Jquery is fast, small, and feature -rich Javascript library. it makes thins like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with easy-to-use API that works across a multitude of browser. With a combination of versatility and extensibility , Jquery has changed the way thay millions of people write Javascript.
